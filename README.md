# Jordan Mosselson - Online CV

This repository contains the source code for Jordan Mosselson's online curriculum vitae (CV). It's a modern, responsive single-page web application designed to showcase his skills, experience, and projects in a clean and engaging manner.

-----

## Features

  * **Responsive Design**: Adapts seamlessly to various screen sizes, from mobile devices to large desktop monitors.
  * **Modern UI**: Utilizes a dark theme with a vibrant color palette, designed for readability and visual appeal.
  * **Dynamic Animations**: Incorporates scroll-triggered animations using `animate.css` to enhance user engagement.
  * **Clear Sections**: Organizes information into well-defined sections: About Me, Work Experience, Education, Technical Skills, and Projects.
  * **Interactive Links**: Provides direct links to Jordan's LinkedIn, GitHub, and email for easy contact.
  * **Font Awesome Icons**: Uses a wide range of icons for improved visual representation and user experience.

-----

## Technologies Used

  * **HTML5**: Structures the content of the CV.
  * **CSS3 (Custom)**: Styles the application, including layout, colors, typography, and responsiveness.
  * **Google Fonts (Poppins)**: Provides a modern and clean typeface.
  * **Animate.css**: A library of ready-to-use CSS animations for adding dynamic effects.
  * **Font Awesome**: Icons for visual enhancement and usability.
  * **JavaScript**: Implements scroll-triggered animations using the Intersection Observer API.

-----

## Setup and Usage

To view this online CV locally, simply follow these steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/JMosselson/Online-CV.git
    ```
2.  **Navigate to the Directory**:
    ```bash
    cd Online-CV
    ```
3.  **Open `index.html`**:
    Open the `index.html` file in your preferred web browser. You can usually do this by double-clicking the file.

-----

## Customization

This CV is designed to be easily customizable.

1.  **Content**: Edit the `index.html` file to update personal details, work experience, education, skills, and projects.
2.  **Styling**: Modify the CSS within the `<style>` tags in `index.html` to change colors, fonts, and layout. The CSS uses custom properties (CSS variables) for easy theme adjustments.
3.  **Animations**: Adjust or add new animations by modifying the `data-anim` and `data-delay` attributes on elements, referring to the `animate.css` documentation for available animations.
